# 💎 Graphico Pro: The Ultimate AI Data Studio

**Graphico Pro** is a professional-grade data analytics and machine learning suite built using **Python** and **Streamlit**. It is designed to empower users to transform raw datasets into actionable insights and predictive models in seconds, without writing a single line of code.

---

## 🚀 Key Features

* **⚡ Instant Visualization:** Seamlessly upload CSV, Excel, or JSON files. Generate high-quality Bar, Line, Scatter, Pie, Histogram, Box Plot, and Area charts with advanced styling options.
* **🧹 Smart Data Cleaning:** Integrated "Mr. Mastermind" logic that automatically detects and handles missing values using Mean and Mode imputation, ensuring the application never crashes.
* **🧠 ML Hub (Intelligence Engine):**
    * **Encoding Tools:** Convert categorical text data into machine-readable formats using Label Encoding and One-Hot Encoding.
    * **AI Forecasting:** Support for Linear Regression and high-degree Polynomial Regression for complex non-linear pattern recognition.
* **🛠️ Data Surgeon:** Manipulate your database directly from the UI—delete columns, remove rows, or update specific values on the fly with a real-time data preview.
* **🎨 Premium UI/UX:** A sleek, Glassmorphism-inspired dark mode interface with custom CSS for a high-end SaaS experience.
* **📊 Analytics Engine:** Get instant descriptive statistics and a raw insight engine to understand your data distribution.
